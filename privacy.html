Privacy Policy

This policy explains how Areospace collects and uses data from users and servers.

1. What We Collect

We collect server IDs, command usage logs, and error messages to improve functionality.

2. How It's Used

Data is used to diagnose bugs, track usage trends, and ensure service reliability.

3. Retention

Data is retained for no longer than 4 weeks unless required for debugging ongoing issues.

4. No Sensitive Data

We do not store or access sensitive personal information without clear user consent.

5. Questions

Contact the developer via the support server if you have questions about your data.
